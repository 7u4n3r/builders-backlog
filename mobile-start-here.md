# Start Here: Mobile Version

Builders Backlog is designed to work even if all you have is a cellphone.

You do not need a laptop.
You do not need a perfect workspace.
You do not need to understand GitHub.

Use this as a simple holding place for ideas, unfinished projects, research, screenshots, links, notes, and half-built things you do not want to lose.

## How To Use This On A Phone

### 1. Copy the template

Open:

`templates/backlog-template.md`

Copy the text.

Paste it into whatever you already use:

- Notes app
- Google Docs
- Gmail draft
- Notion
- Samsung Notes
- Apple Notes
- Text message to yourself
- Private Discord channel
- Any place you can find again

## 2. Name the idea

Do not overthink it.

Examples:

- Podcast idea
- Website rebuild
- Product I might sell
- Research rabbit hole
- App idea
- Court paperwork sorter
- Content series
- Song concept
- Business idea

## 3. Mark the state

Choose one:

- Raw idea
- Needs research
- Started
- Needs cleanup
- Ready to post
- Ready to sell
- Save for later

## 4. Add what already exists

Put links, screenshots, notes, filenames, or reminders in one place.

The goal is not to make it perfect.

The goal is to stop losing the thread.

## 5. Pick one next move

Only choose one.

Examples:

- Find the screenshot
- Write the title
- Make a folder
- Record a voice note
- Add the Gumroad link
- Make the first post
- Move it into GitHub
- Delete what no longer matters

## Phone Rule

If the system takes more energy than the idea, simplify the system.

Builders Backlog is not meant to become another place where projects go to die.

It is meant to help you find the next usable move.
